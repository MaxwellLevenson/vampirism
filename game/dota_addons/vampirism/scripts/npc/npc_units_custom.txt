  //=================================================================================//
  // Vampirism Naming Convention - Please read before making changes
  //
  //   Custom units should follow the following convention.
  //   class_type_tier 
  //   - type is optional if there is only one in a class, as in slayer_tavern.
  //   - tier is optional if there is only one in a tier, as in tower_frost
  //
  //   Examples of good naming:
  //
  //   tower_pearls_t1
  //   - unit class is tower, type is pearls and tier is t1
  //
  //   house_t1
  //   - unit class is house, type is not needed and tier is t1.
  //
  //   Examples of bad naming:
  //
  //  shiny_tower_pearls
  //   - named according to its unit name, not really the best way to do this.       
  //
  //   To indicate a units real name / the name that is shown to the player, place   
  //   it in the ascii header above the unit.
  //=================================================================================//
